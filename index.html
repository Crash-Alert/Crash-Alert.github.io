What is Crash Alert?

Crash Alert is next generation in Plane Crash assists. Inspired by the recent plane crashes and losses, Crash Alert uses the built in accelerometer of the phone to get the force of the shaking of the phone. We then used the geolocation api of android to get the latitude longitude and altitude of the phone in the plane. When the force values exceed a threshhold, we send the data to a syncano server that stores the data for visibility. Rescue workers can then view this data to help find lost planes at the coordinates given.
